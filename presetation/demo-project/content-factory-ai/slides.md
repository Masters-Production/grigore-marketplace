# Content Factory cu AI - Slide-uri Complete Marathon (4 Zile)

## Overview

- **Total slide-uri:** ~180 (45/zi)
- **Durata:** 60-90 min/zi
- **Timp mediu/slide:** 20-30 secunde
- **Format:** Mobile-first (80%+ vizionari de pe telefon)

## Visual Assets Recommendations

- **Paleta de culori:**
  - Primary: #1E3A8A (Deep Blue - incredere, profesionalism)
  - Secondary: #10B981 (Emerald Green - crestere, succes)
  - Accent: #F59E0B (Amber - energie, urgenta)
  - Background: #F8FAFC (Light gray) / #0F172A (Dark mode)

- **Fonturi:**
  - Headline: Inter Bold sau Poppins Bold (min 36pt)
  - Body: Inter Regular (min 24pt)

- **Stil vizual:**
  - Imagini reale, nu ilustratii generice
  - Iconite simple, line-style
  - Screenshots reale din Claude Code, n8n
  - Contrast ridicat pentru mobile

- **Mood:** Profesional dar energic, tech dar accesibil
