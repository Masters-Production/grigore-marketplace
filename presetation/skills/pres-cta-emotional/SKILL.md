# CTA & Emotional Selling Skill

## Overview

This skill enhances the presentation script with advanced emotional triggers, psychological persuasion techniques, and powerful call-to-action sequences based on Hristosenko methodology.

## Input Required

- `script.md` - Base script from pres-content
- `brainstorming.md` - Audience profile

## Core Psychology (From Hristosenko)

### The Emotional Decision Process
**"Люди принимают решения эмоционально, а потом оправдывают их логически"**
(People make decisions emotionally, then justify them logically)

Therefore:
1. First trigger emotion
2. Then provide logical justification
3. Create urgency to act NOW

### The Four Value Depths
When presenting benefits, go deeper:

1. **ИМЕТЬ (To Have)** - Surface level
   - "You'll get access to 12 modules"

2. **ДЕЛАТЬ (To Do)** - Action level
   - "You'll be able to create funnels"

3. **ЧУВСТВОВАТЬ (To Feel)** - Emotional level
   - "You'll feel confident in your marketing"

4. **БЫТЬ (To Be)** - Identity level
   - "You'll become a successful entrepreneur"

**Always push to the deepest level possible.**

## Emotional Trigger Techniques

### 1. Pain Amplification

**Formula:**
```
Current pain → Future consequences → Lost opportunities → Emotional cost

"Сейчас вы [current problem]..."
"Если ничего не изменить, через год вы будете [worse situation]..."
"И всё это время вы будете упускать [opportunities]..."
"Представьте как вы будете чувствовать себя зная что [emotional cost]..."
```

### 2. Desire Amplification

**Formula:**
```
Current desire → Vivid future → Emotional state → Identity shift

"Вы хотите [desire]..."
"Представьте через 3 месяца вы [vivid description]..."
"Как вы будете чувствовать себя когда [emotional state]..."
"Вы станете [new identity]..."
```

### 3. Fear of Missing Out (FOMO)

**Elements:**
- Limited time: "Только до [deadline]"
- Limited spots: "Только [X] мест"
- Exclusive access: "Только для участников"
- Price increase: "После [time] цена будет [higher price]"

### 4. Social Proof Stacking

**Types to include:**
- Results: "[X] человек уже получили [result]"
- Real-time: "[NAME] только что присоединился"
- Similar people: "Такие же как вы [description] уже..."
- Authority: "[Expert/Media] говорят о..."

### 5. Future Pacing

**Formula:**
```
"Представьте..." + vivid sensory details + emotional payoff

"Представьте себе утро через 3 месяца...
Вы просыпаетесь, проверяете телефон...
И видите что ночью пришло [X] новых клиентов...
Как вы будете чувствовать себя в этот момент?"
```

## CTA Enhancement Techniques

### The 15+ CTA Rule
**Every CTA must be unique with a different emotional hook:**

**CTA Types:**

1. **Benefit CTAs:**
   - "Нажмите чтобы получить [specific benefit]"

2. **Fear CTAs:**
   - "Нажмите пока не поздно"
   - "Не упустите возможность"

3. **Social CTAs:**
   - "Присоединяйтесь к [X] людям которые уже..."

4. **Urgency CTAs:**
   - "Нажмите сейчас - осталось [X] минут"

5. **Identity CTAs:**
   - "Нажмите если вы готовы стать [identity]"

6. **Logic CTAs:**
   - "Это просто математика - нажмите"

7. **Guarantee CTAs:**
   - "Нажмите без риска - есть гарантия"

## Objection Handling Scripts

### Pre-emptive Objection Handling

**"I don't have time"**
```
"Я знаю о чём вы думаете - у меня нет времени...
Но подумайте - сколько времени вы ТЕРЯЕТЕ каждый день
делая [ineffective thing]?
Этот курс сэкономит вам [X] часов в неделю."
```

**"I've tried before"**
```
"Возможно вы пробовали раньше и не получилось...
Но дело не в вас - дело в методе.
У меня другой подход: [brief difference]
Поэтому мои студенты получают результат."
```

**"It's too expensive"**
```
"Я понимаю - [price] кажется большой суммой...
Но давайте посчитаем:
- Без этого вы потеряете [X] в год
- С этим вы заработаете [Y]
- Разница: [Z]
Это не расход - это инвестиция."
```

**"I need to think"**
```
"Конечно, подумайте...
Но помните - эта цена только до [deadline]
И я не смогу сохранить её для вас позже.
Если вы хотите [result], действуйте сейчас."
```

## Urgency & Scarcity Enhancement

### Time-Based Urgency
```
"Специальная цена действует только [X часов/до конца вебинара]"
"После [time] цена вырастет до [higher price]"
"Бонусы исчезнут через [X]"
```

### Quantity-Based Scarcity
```
"Я беру только [X] человек в этот поток"
"Осталось [Y] мест"
"[Z] человек уже записались пока мы говорим"
```

### Exclusive Access
```
"Это предложение только для участников сегодняшнего [event]"
"Оно никогда не повторится в таком виде"
"Вы не найдёте это на моём сайте"
```
