# CTA (Call-to-Action) Examples & Templates

## CTA Types

### 1. Primary CTA - The Main Offer
Direct invitation to purchase

### 2. Soft CTA - Engagement
Get them to take a small action first

### 3. Urgency CTA - Time-Based
Create deadline pressure

### 4. Scarcity CTA - Quantity-Based
Limited availability

### 5. Risk Reversal CTA - Guarantee-Focused
Remove fear of loss

---

## Primary CTA Templates

### Template A: Direct Invitation
```
[RU]
"Нажмите кнопку ниже и присоединяйтесь к [PROGRAM NAME] прямо сейчас.
Вы получите мгновенный доступ ко всем материалам."

[EN]
"Click the button below and join [PROGRAM NAME] right now.
You'll get instant access to all materials."
```

### Template B: Transformation Focus
```
[RU]
"Если вы готовы [TRANSFORMATION], нажмите кнопку и начните свой путь сегодня."

[EN]
"If you're ready to [TRANSFORMATION], click the button and start your journey today."
```

### Template C: Problem-Solution
```
[RU]
"Хватит [PROBLEM]. Нажмите кнопку и получите [SOLUTION] уже через 5 минут."

[EN]
"Stop [PROBLEM]. Click the button and get [SOLUTION] in just 5 minutes."
```

## CTA Placement Strategy

### During Content
- Soft CTAs every 15-20 minutes
- Keep engagement high
- Build micro-commitments

### Transition to Offer
- Summarize value delivered
- Identify the gap
- First mention of solution

### Offer Presentation
- Primary CTA after revealing offer
- Repeat after each objection handled
- Urgency CTA near end

### Closing
- Scarcity CTA
- Final urgency
- Risk reversal for hesitant

## CTA Enhancement Techniques

### 1. Visual Emphasis
- Arrow pointing to button
- Button animation
- Countdown timer visible

### 2. Social Proof Stack
```
"[X] человек уже нажали кнопку...
[NAME] только что присоединился...
Не оставайтесь позади."
```

### 3. Future Pacing
```
"Представьте - через [X недель] вы уже [RESULT].
Всё начинается с одного клика."
```

### 4. Cost of Inaction
```
"Каждый день без [SOLUTION] стоит вам [COST].
Прекратите терять [X]. Нажмите сейчас."
```
