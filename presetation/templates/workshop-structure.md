# Workshop Structure Template (2-4 Hours)

## Overview

A workshop is an intensive, hands-on session that delivers a specific transformation while positioning a larger offer.

| Phase | Duration | Focus |
|-------|----------|-------|
| Opening | 15-20 min | Hook + Setup |
| Teaching | 60-90 min | Core Content |
| Implementation | 45-60 min | Hands-On Work |
| Offer | 20-30 min | Present Solution |
| Q&A + Close | 15-20 min | Handle Objections |

## Workshop Types

### Type A: Skill Workshop
Focus on teaching one specific skill
Best for: Technical audiences, B2B

### Type B: Strategy Workshop
Focus on planning/strategy creation
Best for: Entrepreneurs, high-ticket

### Type C: Transformation Workshop
Focus on mindset + action
Best for: Personal development, coaching

## Phase 1: Opening (15-20 min)

### Elements
```
[WELCOME - 3 min]
- High energy greeting
- Thank them for investing time
- Set expectations

[HOOK - 5 min]
- Provocative statement
- "By the end of this workshop you will..."
- Create excitement

[CREDIBILITY - 5 min]
- Your background (brief)
- Results you've achieved
- Student results

[LOGISTICS - 5 min]
- How workshop works
- Materials needed
- When to ask questions
- Stay until the end promise
```

### Key Scripts
```
"Этот воркшоп изменит ваш подход к..."
"This workshop will change your approach to..."

"У вас будет готовый [результат] к концу..."
"You'll have a ready [result] by the end..."

"Оставайтесь до конца - там самое важное..."
"Stay until the end - that's where the most important part is..."
```

## Phase 2: Teaching (60-90 min)

### Structure: 3 Teaching Blocks

Each block (20-30 min):
1. Concept explanation
2. Framework/System
3. Example/Demo
4. Bridge to next block

### Block 1: Foundation
```
[CONCEPT - 10 min]
- Core idea explained simply
- Why this matters
- Common misconceptions

[FRAMEWORK - 10 min]
- Step-by-step system
- Visual diagram
- Easy to remember

[EXAMPLE - 10 min]
- Real case study
- Before/After
- Belief shift moment
```

### Block 2: Deep Dive
```
[ADVANCED CONCEPT - 10 min]
- Building on Block 1
- Nuances and details
- Pro tips

[IMPLEMENTATION GUIDE - 10 min]
- How to actually do it
- Tools needed
- Common mistakes to avoid

[DEMONSTRATION - 10 min]
- Live example
- Your process
- Behind the scenes
```

### Block 3: Mastery
```
[SECRET TECHNIQUE - 10 min]
- The "insider" method
- What separates good from great
- Advanced strategy

[CASE STUDY - 10 min]
- Success story
- Challenge overcome
- Results achieved

[TRANSITION - 5 min]
- Recap all three blocks
- What's still missing
- Bridge to implementation
```

## Phase 3: Implementation (45-60 min)

### Purpose
- Create immediate value
- Build investment
- Generate "aha" moments
- Prepare for offer

### Structure
```
[SETUP - 5 min]
- Explain the exercise
- Materials/templates
- Expected outcome

[GUIDED WORK - 30 min]
- Step-by-step guidance
- Check-ins every 10 min
- Answer questions live
- Celebrate progress

[SHARE + REVIEW - 15 min]
- Volunteers share work
- Feedback and praise
- Common patterns
- Next steps preview

[GAP IDENTIFICATION - 5 min]
- What they achieved
- What's still needed
- "To get full results..."
```

### Implementation Scripts
```
"Сейчас вы создадите свой собственный..."
"Now you'll create your own..."

"У вас есть 10 минут. Начинайте!"
"You have 10 minutes. Begin!"

"Кто хочет поделиться результатом?"
"Who wants to share their result?"
```

## Phase 4: Offer (20-30 min)

### Structure
```
[BRIDGE - 3 min]
- Celebrate their work
- Show what's possible
- Introduce solution

[PROGRAM REVEAL - 10 min]
- Name and promise
- What's included
- Module breakdown (outcomes)
- Bonuses

[VALUE STACK - 5 min]
- Total value calculation
- Compare to alternatives
- Investment reveal

[GUARANTEE + CTA - 5 min]
- Risk reversal
- Clear next step
- Urgency elements

[SPECIAL OFFER - 5 min]
- Workshop-only pricing
- Fast-action bonus
- Deadline
```

### Offer Scripts
```
"То что вы создали - это начало. Полная система включает..."
"What you created is the beginning. The full system includes..."

"Только для участников воркшопа - специальная цена..."
"Only for workshop participants - special price..."

"Это предложение действует только 48 часов..."
"This offer is only valid for 48 hours..."
```

## Phase 5: Q&A + Close (15-20 min)

### Structure
```
[OPEN Q&A - 10 min]
- Answer questions
- Handle objections
- Repeat CTA after each answer

[FINAL PUSH - 5 min]
- Recap transformation
- Last success stories
- Deadline reminder

[CLOSE - 5 min]
- Thank participants
- Remind of offer
- Clear next step
- Stay for bonus content (for buyers)
```

## Workshop-Specific Elements

### Materials
- Workbook/Worksheet (PDF)
- Templates
- Checklists
- Resource list

### Engagement Techniques
- Polls every 15-20 min
- Chat interaction
- Hand raises
- Breakout rooms (if platform allows)

### Follow-Up
- Recording access
- Completed worksheet
- Email sequence (3-5 emails)
- Deadline reminders

## Key Metrics

- Attendance rate: 40-60% of registrants
- Stay rate: 70%+ to offer
- Conversion rate: 10-25%
- Average order value: Mid-ticket optimal
