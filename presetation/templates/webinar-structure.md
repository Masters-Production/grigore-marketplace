# Webinar Structure Template (60-90 minutes)

## Timing Overview

| Section | Duration | Purpose |
|---------|----------|---------|
| Opening | 5-10 min | Hook + Credibility |
| Problem | 10-15 min | Pain Agitation |
| Content | 25-35 min | Value + Belief Shifts |
| Transition | 5 min | Bridge to Offer |
| Offer | 15-20 min | Present + Stack |
| CTA | 5-10 min | Close + Urgency |

## Section 1: Opening (5-10 min)

### Elements
1. **Attention Hook** - Provocative statement or question
2. **Big Promise** - What they'll learn/achieve
3. **Credibility** - Why listen to you
4. **Roadmap** - What's coming

### Script Framework
```
[HOOK - 30 sec]
"Что если я скажу вам, что..."
"What if I told you that..."

[PROMISE - 1 min]
"За следующие 60 минут вы узнаете..."
"In the next 60 minutes you'll discover..."

[CREDIBILITY - 2 min]
Results you've achieved
Students' results
Your story (brief)

[ROADMAP - 1 min]
"Мы рассмотрим 3 ключевых момента..."
"We'll cover 3 key points..."
```

## Section 2: Problem Agitation (10-15 min)

### Elements
1. **Identify Pain** - Name their struggle
2. **Amplify** - Show consequences of not solving
3. **Common Mistakes** - Why they failed before
4. **Hope** - There is a solution

### Framework
```
[IDENTIFY - 3 min]
"Знакомо ли вам когда..."
"Do you know the feeling when..."

[AMPLIFY - 5 min]
"А что будет если так продолжать..."
"And what happens if this continues..."

[MISTAKES - 5 min]
"Большинство делают эти 3 ошибки..."
"Most people make these 3 mistakes..."

[HOPE - 2 min]
"Но есть другой путь..."
"But there's another way..."
```

## Section 3: Content (25-35 min)

### Structure: 3 Content Blocks

Each block:
- Teaching point
- Example/Story
- Belief shift
- Bridge to product

### Content Block Framework
```
[TEACH - 5 min]
Explain concept simply

[EXAMPLE - 3 min]
Case study or story

[BELIEF SHIFT - 2 min]
"Возможно вы думали что... На самом деле..."
"You might have thought... Actually..."

[BRIDGE - 1 min]
"Это один из элементов моей системы..."
"This is one element of my system..."
```

## Section 4: Transition to Offer (5 min)

### Elements
1. **Summary** - Recap what they learned
2. **Gap** - What's still missing
3. **Invitation** - Introduce the solution

### Framework
```
[SUMMARY - 2 min]
"Итак, вы узнали..."
"So you've learned..."

[GAP - 2 min]
"Но чтобы получить результат нужно ещё..."
"But to get results you also need..."

[INVITATION - 1 min]
"Поэтому я хочу пригласить вас..."
"That's why I want to invite you..."
```

## Section 5: Offer Presentation (15-20 min)

### Elements
1. **Product Introduction** - Name + Big Promise
2. **Module Breakdown** - What's included
3. **Value Stack** - Build perceived value
4. **Bonuses** - Add extras
5. **Price Reveal** - With anchoring
6. **Guarantee** - Risk reversal

### Framework
```
[INTRO - 2 min]
Name the program
State the transformation

[MODULES - 8 min]
Module 1: [Result]
Module 2: [Result]
Module 3: [Result]
(Focus on outcomes, not features)

[STACK - 3 min]
"Всё это стоит X, но вы получите..."
"All this is worth X, but you'll get..."

[BONUSES - 3 min]
Bonus 1: [Value]
Bonus 2: [Value]
"Только для участников вебинара..."
"Only for webinar participants..."

[PRICE - 2 min]
Anchor high
Reveal actual price
Show savings

[GUARANTEE - 2 min]
Risk reversal
"Если не получите результат..."
"If you don't get results..."
```

## Section 6: CTA + Close (5-10 min)

### Elements
1. **Clear Action** - What to do now
2. **Urgency** - Why now
3. **Scarcity** - Limited availability
4. **Final Push** - Last benefits
5. **Q&A Transition** - Handle objections

### Framework
```
[ACTION - 1 min]
"Нажмите кнопку ниже..."
"Click the button below..."

[URGENCY - 2 min]
"Цена действует только до..."
"This price is only valid until..."

[SCARCITY - 2 min]
"Осталось только X мест..."
"Only X spots remaining..."

[FINAL PUSH - 2 min]
Recap transformation
Paint the future
"Представьте через 3 месяца..."
"Imagine in 3 months..."

[Q&A - ongoing]
Answer questions
Repeat CTA
Handle objections
```

## Key Metrics

- Target conversion: 5-15%
- Attendance rate: 30-50% of registrants
- Watch time: 60%+ should stay to offer
- Click rate on CTA: 20-30%
